What is Alicht CSS?

Some modern, yet basic, CSS bits we put together so as to set a nice, shady and somewhat classy ambience.

This is a live demo. It should help you see what alicht supports already, and get to know how it works.

Get started

Alicht CSS is part of the Archost project. As such, it is currently a Work-in-Progress project.

Check out its repository and help us enhance it! And if you're using it, remember to regularly check for updates :)
