Buttons

Buttons have styles, states, and sizes variations.
Add class="button" to any <a> link element to turn it into a button.

<button> elements get the button style as well, without any class.

Styles

Button Primary Success Error Warning

.button, .button.primary, .button.success, .button.error, .button.warning.

States

Normal Disabled

.button, .button.disabled.

Sizes

Smaller Small Default Big Bigger

.button.smaller, .button.small, .button.big, .button.bigger.
