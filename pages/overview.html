Overview

(Very) Modern

Alicht makes use of several of the newest features that are now part of CSS. That includes flexboxes, CSS transforms, CSS animations, @media queries, and also a few minor properties.

Because of that approach, please do note Alicht will only work with the most up to date browsers out there.

Keeping it simple.

I try not to get fancy when designing Alicht. That stands both for its design and for its features.

I want Alicht to be a basic CSS framework out of the box. That means it will never pack as many components and layout possibilities as other popular frameworks.

Alicht's main purpose is to try keep a balance between basic functionality, basic modularity, and a good appearance.

Open. Of course.

Alicht is on Github, so you probably guessed it already. What's important though is that I invite you to report any issue/bug you might encounter using Alicht. Suggestions and ideas for improvements are welcome as well.
